# Project_5
Agricultural Product Analysis

I built this Power BI dashboard to analyze how climate conditions and farming practices influence crop yield.

📊 Key Insights

● Irrigation methods affect productivity, with sprinkler and manual systems showing slightly better performance.

● Rainfall and soil moisture support stable yields under consistent conditions.

● Soybean and wheat recorded relatively higher yields across crop types.

● Organic and mixed fertilizers perform competitively with inorganic options.

● Yield peaks around June–July, reflecting seasonal harvest patterns.

● Garissa County shows the highest average yield among regions.

🛠️ Tools

Power BI • DAX • Power Query • Data Analysis
